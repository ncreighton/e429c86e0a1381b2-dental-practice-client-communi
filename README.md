## Installation

### From VS Code Marketplace
1. Open VS Code
2. Press `Ctrl+P` (or `Cmd+P` on Mac)
3. Type `ext install digitalforge.e429c86e0a1381b2`
4. Press Enter

### From VSIX file
1. Download the `.vsix` file
2. In VS Code: `Extensions > ··· > Install from VSIX`
3. Select the downloaded file

## Usage

Press `Ctrl+Shift+P` and search for `Dental Practice Client Communication Portal Plugin` to see all available commands.

**Keyboard shortcut:** `Ctrl+Shift+Alt+A` (or `Cmd+Shift+Alt+A` on Mac)
